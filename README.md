 # Mobila la Comanda Website

## Description
This project is a furniture presentation website developed for a custom furniture business. It was created to provide a professional online presence and support future business promotion.

## Project Status
Portfolio / business website project currently hosted on GitHub.

## Features
- Service presentation website
- Structured content for clear business communication
- Responsive layout for desktop and mobile devices
- Clean and simple user interface
- Lightweight static website

## Technologies
- HTML5
- CSS3
- JavaScript

- Implemented styling and responsive behavior
- Worked on usability and visual improvements
- Managed version control and updates via GitHub

## Purpose
This project demonstrates practical experience in building business-oriented websites and translating real-world requirements into a functional web solution.
